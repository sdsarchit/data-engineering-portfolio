Project Report Key Summary

Key findings, ensemble classifiers, and datasets from your academic BVRIT dissertation.

Abstract & CICIDS2017 Dataset

Network Intrusion Detection Systems (NIDS) are critical tools for administrators to detect security breaches. This project implements a machine learning-based approach using ensemble classifiers.

The **CICIDS2017** dataset was used to validate the model, capturing benign traffic and seven common attack categories: DDoS, DoS, Botnet, Web Attack, PortScan, FTP-Patator, and SSH-Patator, containing over 80 traffic flow features.

Algorithms & Ensemble Paradigm

Existing single-algorithm methods (like SVM) are slower, less stable, and suffer from high false alarm rates on large-scale datasets. This project combines:

  • XGBoost: Gradient boosting framework with L1/L2 regularization to prevent overfitting.
  • Random Forest: Bagging meta-estimator to decrease prediction variance.
  • Extra Trees: Randomized split nodes to counter correlation errors.
  • Voting Classifier: Hard voting ensemble combining individual model outputs.

Key Model Accuracy Comparisons

The ensemble classifiers outperformed traditional single-model algorithms in both detection rates and computation speeds:

  • XGBoost achieved the highest individual detection accuracy of 99.91%.
  • Random Forest Classifier achieved an accuracy of 99.89%.
  • Extra Trees Classifier achieved an accuracy of 99.81%.
  • Ensemble Voting Classifiers achieved up to 99.89% accuracy while dramatically reducing false negative rates across highly imbalanced attack classes.
